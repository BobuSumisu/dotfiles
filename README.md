# dotfiles

## Installation

    $ git clone https://github.com/BobuSumisu/dotfiles.git 
    $ cd dotfiles
    $ ./provision.sh

## Overview

### GNU/Linux 

* `~/.bashrc` 
* `~/.bash_aliases`
* `~/.Xresources`

### Vim

* `~/.vimrc`
* `~/.vim`

### i3

* `~/.i3/config`
* `~/.i3status.conf`

### Scripts

* `~/.bin/`
