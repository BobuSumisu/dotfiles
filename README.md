# dotfiles

## Installation

    $ git clone https://github.com/BobuSumisu/dotfiles.git 
    $ cd dotfiles
    $ ./provision.sh

## Overview

### Meta

* `provision.sh`

### GNU/Linux 

* `~/.bashrc` 
* `~/.bash_aliases`
* `~/.Xresources`
* `~/.config/user-dirs.dirs` 

### Vim

* `~/.vimrc`
* `~/.vim`

### i3

* `~/.i3/config`
* `~/.i3status.conf`
* `~/.config/dunst/dunstrc`

### Scripts

* `~/.bin/`
