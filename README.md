# dotfiles

## Installation

    $ wget https://raw.githubusercontent.com/BobuSumisu/dotfiles/master/boot.sh
    $ ./boot.sh

## Overview

### GNU/Linux 

* `~/.bashrc` 
* `~/.bash_aliases`
* `~/.Xresources`

### Vim

* `~/.vimrc`
* `~/.vim`

### i3

* `~/.i3/config`
* `~/.i3status.conf`

### Scripts

* `~/.bin/`
